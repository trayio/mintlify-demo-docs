# Overview

## Overview

Iterable empowers growth marketers to create world-class user engagement campaigns throughout the full lifecycle, and across all channels.

## Authentication

Log into the Iterable app. From the sidebar menu, select API Keys under Integrations. Click on Create new api key and select a **Standard** key type. You will then be able to create a new API key, which you will paste in the Tray authentication window.
