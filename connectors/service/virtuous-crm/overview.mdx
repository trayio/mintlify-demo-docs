# Overview

## Overview

The tray.io Virtuous CRM connector allows you to create, update, delete, archive, get and list contacts.

## Authentication

When using the Virtuous CRM connector, the first thing you will have to do is click on 'New Authentication' in the step editor:

Ensure you correctly add your Virtuous username/email and password correctly otherwise operations will not be able to be ran correctly.
