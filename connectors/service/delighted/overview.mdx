# Overview

## Overview

Delighted is** **a self-serve experience management platform. Get employee, product, and customer feedback in minutes.

## **API Information **

The Base URL used for the Delighted connector is **https://api.delighted.com/v1**. More information can be found on their main [API documentation (v1)](https://app.delighted.com/docs/api/sending-to-people) site.

## Authentication

Within the builder, click on the Delighted connector to display the connector properties panel. Select the 'Auth' tab and click the 'New authentication' button.

In the Tray.io authentication pop-up modal, name the authentication in a way that will quickly identify it within a potentially large list. For example, whether it is a Sandbox or Production auth, etc.

Consider who/ how many people will need access to this authentication when choosing where to create this authentication ('Personal' or 'Organisational').

The second page asks you for your Delighted **API key**.

To get these fields, head to the main dashboard within your Delighted account. Click on the **Integrations** option in the top menu.

Select API from the integration options presented.

Within the main text you will see your **API key**. Copy and paste it into your Tray.io authentication modal from earlier.

Once you have added these fields to your Tray.io authentication pop-up window, click the 'Create authentication' button.Â 

Your connector authentication setup should now be complete.
