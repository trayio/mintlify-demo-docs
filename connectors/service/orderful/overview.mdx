# Overview

## Overview

Orderful  is a modern EDI platform.  It is the fastest way to trade EDI. Go live with new trading partners in days, not months, with Orderful’s modern EDI software.

## **API Information **

The Orderful connector has the following Base URL: **https://api.orderful.com/v3**. More information can be found on their main[ API documentation (v3) ](https://docs.orderful.com/reference/overview)site. Details regarding their API limitations can be found [here](https://docs.orderful.com/reference/best-practices#rate-limits). 

## Authentication

Within the builder, click on the Orderful connector to display the connector properties panel. Select the **Auth** tab and click on the **New authentication** button.

In the Tray.io authentication pop-up modal name your authentication in a way that will quickly identify it within a potentially large list. For example whether it is a Sandbox or Production auth, etc.

Consider who/ how many people will need access to this authentication when choosing where to create this authentication ('Personal' vs 'Organisational').

The next page asks you for your **API Key** credentials.

To get these fields, head to the Orderful dashboard. Click on the settings.

Here click on the API credentials page.  Then click on show button to see the API token 

Once you have added these fields to your Tray.io authentication pop-up window click the **Create authentication** button. 

Your connector authentication setup should now be complete. Please run the simplest operation available to test and make sure you can retrieve data as expected.
