# Overview

## Overview

HubSpot offers a full stack of software for marketing, sales, and customer service, with a CRM at its core. They can be used independently or together.

## API Information 

The Base URL used for the HubSpot connector is** https://api.hubapi.com**. More information can be found on their main [API documentation sites](https://developers.hubspot.com/docs/api/overview). This is where users will also be able to find the [API Limitations](https://developers.hubspot.com/docs/api/usage-details) page.
